# Community Helpers Explorer

A MERN stack educational app to teach children with autism about community helpers using a visual-first, accessible approach.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)

## Features
- Autism-friendly UI (pastel palette, large buttons, no intrusive animations)
- Helper Gallery, Role Pages, and Matching Game
- Text-to-Speech for all descriptions
- Responsive and accessible design

## Setup

### Backend
1. `cd backend`
2. `npm install`
3. Add your MongoDB URI to `.env`
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

---

Replace image and audio assets in `/public` as needed for your deployment.
