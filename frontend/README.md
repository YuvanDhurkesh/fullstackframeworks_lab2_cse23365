# Community Helpers Explorer Frontend

This is the React.js frontend for the Community Helpers Explorer MERN app.

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Features
- Autism-friendly, high-contrast, pastel UI
- Helper Gallery, Role Pages, and Matching Game
- Text-to-Speech for all descriptions
- Responsive and accessible design
